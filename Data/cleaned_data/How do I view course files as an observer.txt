How do I view course files as an observer
Open Course
In Global Navigation, click the Courses link, then click the name of the course.
Open Course Files
In Course Navigation, click the Files link.
Open a Folder
To open folder contents in the preview pane on the right-hand side, click an unlocked folder from the left-hand side.
Open File
Click the name of the file.
Preview File
The preview displays the file and file information details, as well as a link to download the file.
You can scroll through the document, zoom in and out, and view the document in full screen.
View Button
You can also preview a file by clicking the View button.
Download Files
Click the Options icon and then click the Download link.
