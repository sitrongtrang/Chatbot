How do I enable self-enrollment in an account and allow students to self-enroll in a course as an admin
Open Account
In Global Navigation, click the Admin link, then click the name of the account.
Open Settings
In Account Settings, click the Settings link. 
Allow Self-Enrollment
In the Settings tab, locate the Allow Self-Enrollment drop-down menu.
Set Self-Enrollment
Select the type of self-enrollment permission you want to enable. 
To enable courses that arenâ€™t linked to any SIS data or affected by SIS imports, click the For Manually-Created Courses option. 
To enable any type of course, whether manually-created or created using the SIS Import tool, click the For Any Courses option.
Update Settings
Click the Update Settings button.
