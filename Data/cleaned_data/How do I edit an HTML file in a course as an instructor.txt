How do I edit an HTML file in a course as an instructor
Open Files
In Course Navigation, click the Files link.
Download HTML File
Click the line item for the file. Click the Downloadicon. Or click the Options icon and then click the Download link. 
Open HTML File
Your file will download to your desktop. Open the appropriate folder to view the file.
Note: The file may be saved to the Downloads folder on your computer.
Edit HTML File
After you open the HTML file, edit the file using a text or HTML editor on your computer. Save the file.
Upload HTML File
In Files, click the Upload button.
Open HTML File
Click the title of the file you want to upload and click the Open button.
Replace Duplicate HTML File
To replace the existing file with the edited file, click the Replace button.
Preview HTML File
Click the name of the file and preview the HTML file.
