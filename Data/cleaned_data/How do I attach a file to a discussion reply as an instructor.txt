How do I attach a file to a discussion reply as an instructor
Open Discussions
In Course Navigation, click the Discussions link.
Open Discussion
Click the name of the discussion you want to participate in.
Create Reply
Create a new discussion entry by clicking the Reply text field.
Attach File
After adding content using the Rich Content Editor to the discussion reply, click the Attach link. 
Note: If you cannot attach a file, this feature has been disabled.
Browse for File
Depending on your browser, click the Choose File or Browse button to locate personal files on your computer.
Select File Upload
In the file window, locate and click the name of the file. Click the Choose or Open button.
Note: You can only upload one attachment in your reply.
Post Reply
Click the Post Reply button.
View Discussion Reply
View your discussion reply. Your attached file will appear under the text.
