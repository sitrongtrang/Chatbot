How do I edit a course page as a student
Open Pages
In Course Navigation, click the Pages link.
View Pages
Pages is designed to open to the front page for the course, if there is a front page selected. Click the View All Pages button to select a page from the Pages Index.
Open Page
Click the title of the page.
Edit Page
Click the Edit button.
Note: You will only see the Edit button if you are allowed to edit the page.
Edit Content
Edit the content on the page using the Rich Content Editor or switch to HTML view to edit.
Save Changes
You can notify users that content has changed by selecting the Notify users that this content has changed checkbox. Click the Save button.
View Page
View your changes.
