How do I select a course template in an account as an admin
Open Settings
Open the account or sub-account where you want to apply the course template. In Account Navigation, click the Settings link.
Select Template
Locate the Course Template settings. By default, no course template is selected at the account level. 
To select a template, click the Template drop-down menu and select the template you want to use.
View Inherited Template
If you are selecting a course template in a sub-account, the template is identified by an Inherit label. To select a unique template for a sub-account, click the name of the template.
Update Settings
Click the Update Settings button.
