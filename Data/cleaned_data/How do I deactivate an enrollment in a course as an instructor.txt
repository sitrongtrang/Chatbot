How do I deactivate an enrollment in a course as an instructor
Open People
In Course Navigation, click the People link.
Locate User
In the search field, search for the name of the user. You can also filter users by role in the Roles drop-down menu.
Deactivate User
Click the user's Options icon, then select the Deactivate User link.
Confirm Deactivation
Click the OK button.
Reactivate User
If you want to re-activate the user at a later time, click the user's Options icon, then select the Re-activate User link.
