How do I see which notifications have been sent to a user in an account as an admin
Open Account
In Global Navigation, click the Admin link, then click the name of the account.
Open Admin Tools
In Account Navigation, click the Admin Tools link.
Open View Notifications
Click the View Notifications tab.
Search for Student 
In the search field, type the name of the student and click the Find button. Click the student's name.
Enter Date Range
Create the date range you want to search. You can enter dates in the From Date and To Date date fields, or you can use the calendar icon to select dates. Then click the Find button.
View Notifications Sent to User
View the notifications sent to the user from Canvas.
