How do I create an outcome for an account in Improved Outcomes Management as an admin
Open Account
In Global Navigation, click the Admin link, then click the name of the account.
Open Outcomes
In Account Navigation, click the Outcomes link.
Create New Outcome
To create a new outcome, click the Create button.
Create Outcome
Enter the name or code for the outcome in the Name field.
To add a friendly name for the outcome, enter a name in the Friendly Name field.
Use the Rich Content Editor to add an outcome description in the Description field.
Modify Mastery Scale
You can make changes to outcome mastery scales in the Mastery tab.
Modify Mastery Calculation Method
You can make changes to calculation methods in the Calculation tab.
View Outcome
View your newly created outcome.
