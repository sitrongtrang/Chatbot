How do I add feedback to a question in New Quizzes as an instructor
Edit Question
Locate the question where you would like to add feedback. Click the Edit icon.
Open Feedback
Click the Feedback icon.
View Feedback
Student feedback can be customized to display feedback for correct answers or incorrect answers. You can also display the same feedback for all answers.
Add Feedback
To add feedback, use the Rich Content Editor to create customized feedback. Your feedback can include images, media, or links.
Save Feedback
To save your feedback, click the Done button.
