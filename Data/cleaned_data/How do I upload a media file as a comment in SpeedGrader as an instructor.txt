How do I upload a media file as a comment in SpeedGrader as an instructor
Open SpeedGrader
Open SpeedGrader from any assignment, graded discussion, or quiz.
Open Student Submission
Use the student list to locate a student submission.
Open Studio Capture
In the Assignment Comments area, click the Video comment icon.
Note: If the assignment is supported by Assignment Enhancements, the Assignment Comments section may display as Comments for this Attempt.
Open Upload Media
Click the Upload Media tab. 
Select Media Type
To upload an audio file, click the Select Audio File button. 
To upload a video, click the Select Video File button.
Select File
Select a file. Then click the Open button.
View Posted Media
The recording is automatically processed, submitted, and displays in the Assignment Comments area.
