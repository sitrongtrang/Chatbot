How do I add a text header as a module item as an instructor
Open Modules
In Course Navigation, click the Modules link.
Add Content
Click the Add Item button.
Select Item Type
In the drop-down menu, select the Text Header option.
Add Text
Type the text you wish to add in the header field. Open the Indentation drop-down menu and select the level of indentation for the text header.
Add Item
Click the Add Item button.
View Module
View the new text header in your module.
