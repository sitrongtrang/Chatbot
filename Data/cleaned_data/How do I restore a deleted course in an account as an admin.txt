How do I restore a deleted course in an account as an admin
Open Account
In Global Navigation, click the Admin link, then click the name of the account.
Open Admin Tools
Click the Admin Tools link.
Select Restore Type
In the Restore Content tab, click the Select a restore type drop-down menu and select the Courses option.
You can also restore deleted users.
Enter Course ID
In the Course ID field, enter the course ID of the course you want to restore. Click the Find button. 
Restore Course
When the course appears, click the Restore button. 
Confirm Course Restore
Canvas will confirm you have restored your course.
To view the course, click the View Course button. To add user enrollments into the course, click the Add Enrollments button.
