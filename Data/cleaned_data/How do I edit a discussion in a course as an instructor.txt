How do I edit a discussion in a course as an instructor
Open Discussions
In Course Navigation, click the Discussions link.
Open Discussion
Click the name of the discussion.
Edit Topic
Click the Edit button. 
Edit Topic in Discussions Redesign
To edit a discussion in Discussions Redesign, click the Options icon and click the Edit option.
Edit Discussion
To edit the discussion, use the Rich Content Editor. You can also edit other discussion options.
Save Discussion
Click the Save button.
Note: If you are editing an unpublished discussion and want to publish it, click the Save & Publish button.
