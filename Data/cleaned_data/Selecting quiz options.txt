Selecting quiz options
No Submission Quiz
If a quiz is taken in a classroom or outside of Canvas, you should create an assignment using the No Submission submission type. This assignment type allows you to manually set the points possible and enter grades for your students. 
Create No Submission Assignment
Create a new assignment in your Canvas course. In the Submission Type drop-down menu, select the No Submission option.
View Assignment in Gradebook
You can view the assignment in the Gradebook and enter student's quiz grades.
