How do I upload a file to a group as a student
Open Files
In Group Navigation, click the Files link.
View Files
Once you click the Files navigation link, you will see where all the files are located for the group.
Add Files
Click the Upload button to add new files to the group.
Choose File to Upload
Find the file you want to upload to the group. Click the Open button once to upload the file.
View Group Files
Once you upload the file you selected, it will show up in the Files section of the group navigation. This way all the group members will be able to access the file.
