How do I attach a media comment to a message as a student
Open Inbox
In Global Navigation, click the Inbox link.
Compose Message
To start a new message, click the Compose icon.
Open Media
To attach a media comment or file, click the Media icon.
Record Media Comment
A pop-up window will appear in your browser. If you see a message from instructure.com requesting access, click the Allow button.
Click the Record tab. Select your preferred media method: record with microphone or record with webcam. When you're ready to record, click the Start Recording button.
Create Recording
The timer with the dot indicates you are recording your media comment, as well as the length of the comment. Once you are finished recording, click the Finish button.
Save Recording
Before you attach the media comment, you have the chance to review it. 
In the file name field, enter a name for your media comment.
If you want to re-record the recording, click the Start Over button.If you are happy with the result, click the Save Media button.
Upload Media Comment 
If you have a previously recorded media file, you can upload it as part of your message. Click the Computer tab. Click or drag and drop a media file to media uploader to upload a file from your computer.
Upload Media File
Locate the media file on your computer. Click the Open button.
Send Message
Your media comment displays in your message. If you want to remove the comment, click the Remove icon to remove it from the message. When you are finished, click the Send button.
View Media Comment
The media comment displays below the message.
