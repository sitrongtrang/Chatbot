How do I view the outcomes results report for an individual student in a course as an instructor
Open People
In Course Navigation, click the People link.
Locate User
In the search field, search for the name of the user. You can also filter users by role in the Roles drop-down menu.
Open User Details
Click the user's Options icon, then select the User Details link.
Open Outcomes Report
Locate the user details sidebar. Depending on the page display, click the Outcomes Report link or button.
View Report
View the report.
