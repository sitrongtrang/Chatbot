How do I record the last day a user attended a course as an instructor
Open People
In Course Navigation, click the People link.
Locate User
In the search field, enter the name of the user. You can also filter users by role in the Roles drop-down menu.
Open User Details
Click the user's Options icon, then select the User Details link.
View Memberships
Locate the Membership Section in the user details page.
If your institution has enabled Profiles, the Membership Section displays directly in the page.
View More User Details
If Profiles is not enabled for your institution, you can view the Membership Section by clicking the more user details link.
Enter Last Date Attended
Click the Last Day Attended field and click the last day the student attended in the calendar. 
You can also type a date directly in the text field.
View Last Date Attended
View the last attended date in the text field.
