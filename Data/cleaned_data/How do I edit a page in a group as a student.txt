How do I edit a page in a group as a student
Open Pages
In Group Navigation, click the Pages link.
View Pages
Pages is designed to open to the front page for a group, if there is a front page selected. Click the View All Pages button to select a page from the Pages Index.
Open Page
Click the title of the page you wish to change.
Edit Page
Click the Edit button to make changes to the page.
Edit Content
Edit the content and add links and media using the Rich Content Editor or switch to the HTML Editor.
Save Changes
Click the Save button.
View Changes
View the changes you made to the page.
