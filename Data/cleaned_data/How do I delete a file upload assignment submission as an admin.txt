How do I delete a file upload assignment submission as an admin
Open Account
In Global Navigation, click the Admin link, then click the name of the account.
Open Course
To find the course in the account, you can use the course filters or search options.
To open a course, click the name of the course you want to open.
Open Assignments
In Course Navigation, click the Assignments link.
Open Assignment
Click the name of the assignment.
Open SpeedGrader
Click the SpeedGrader link.
Delete Submission
Find the submission file link. Then click the Delete icon.
Confirm Deletion
To confirm you want to delete the submission file, click the OK button.
View Deleted File
After the file is deleted, a message displays in SpeedGrader to inform instructors that the file was deleted. The submission file is also replaced by File_Removed.pdf.
