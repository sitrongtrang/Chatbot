How do I edit module items as an instructor
Open Modules
In Course Navigation, click the Modules link.
Edit Module Item
Click the Options icon and select the Edit link.
Edit Item Details
You can edit the item title by typing in the title field. You can also edit the indentation level by selecting the indent drop-down menu.
Note: Changes made to one item will be made to all iterations of the item.
Update Item
Click the Update button.
View Module Item
View the changes to the Module item.
