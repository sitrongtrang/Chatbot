How do I send a message to a user in a student group in the Inbox as an instructor
Open Inbox
In Global Navigation, click the Inbox link.
Compose Message
Click the Compose icon.
Select Course
To send a message to a student course group, click the Course drop-down menu.
Open Student Groups
To view student groups in the selected course, click the Address Book icon, then click the Student Groups link. 
Open Group Roster
To view the student group roster, click the name of the student group.
Add Users
Select the name of the student you'd like to message. 
To message all students in the student group, click the All in Student Group link.
Send Message
Enter a subject for your message in the Subject field.
If you are sending your message to multiple users, but do not want users to see who else was included in the message, click the Send an individual message to each recipient checkbox. If your message includes over 100 recipients (including you as the sender), this checkbox will be selected by default.
Type your message in the Message field. All content is sent in plain text. Any URLs included in your message automatically become clickable after the message is sent.
To add an attachment, click the Attachment icon.
To add a media file, click the Media File icon.
To send your message, click the Send button.
