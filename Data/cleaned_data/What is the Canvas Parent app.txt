What is the Canvas Parent app
View Student Courses and Grades
The Courses tab displays the list of courses that their students are enrolled in. For each course, parents can see course grades, syllabus, assignments, and events. Grades display according to the course grading scheme. 
View Student Assignments and Events
Assignments and events can be viewed by course or week. Parents can set reminders for a specific assignment or event, which are sent to mobile devices as push notifications.
Set Alerts for Student Activity
Parents can set alerts for specific assignment and grade activity. Alerts are displayed in the Alerts page in the app.
