How do I mark a module item as done for a module requirement as a student
Open Modules
In Course Navigation, click the Modules link. 
Open Content Page
Click the name of the content page with the requirement.
Mark Done
Click the Mark as Done button. 
Confirm Done
When you mark the page as done, the button changes from Mark as Done to Done. 
Unmark Page
If you want to unmark the page as done, click the Done button. The button will turn back into the Mark as Done button.
