How do I attach a file to an announcement in a course as an instructor
Open Announcements
In Course Navigation, click the Announcements link.
Open Announcement
Click the name of the announcement.
Edit Announcement
Click the Edit button. 
Attach File
Click the Choose File button.
Browse for File
Choose the file you want to attach. To attach the file to the Announcement, click the Open button. 
Save Announcement
Click the Save button.
View Announcement
View the attachment in the announcement.
