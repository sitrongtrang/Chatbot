How do I view annotation feedback comments directly in an assignment submission as an observer
Open Grades
In Course Navigation, click the Grades link.
Open Assignment Submission Details
Locate the assignment and click the assignment name.
Open Feedback
Click the View Feedback button.
Note: If the assignment attachment displays a Preview button, the file is not DocViewer compatible.
View Annotated Comments
View the instructor's annotated comments. When an annotated comment includes multiple content lines, the content box displays an ellipses link. To expand a comment, click the Ellipses link.
Download File
To download a PDF of the submission and annotations, click the Download icon.
