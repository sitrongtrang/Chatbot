How do I restore a deleted user in an account as an admin
Open Account
In Global Navigation, click the Admin link, then click the name of the account.
Open Admin Tools
Click the Admin Tools link.
Select Restore Type
In the Restore Content tab, click the Select a restore type drop-down menu and select the Users option.
You can also restore deleted courses.
Search User ID
In the User ID field, enter the user ID of the user you want to restore. Click the Find button.
Restore User
When the user is displayed, click the Restore button.
Confirm User Restore
Canvas confirms you have restored your user.
To view the user, click the View User button.
