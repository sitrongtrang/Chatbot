How do I edit user roles in a course as an instructor
Open People
In Course Navigation, click the People link.
Locate User
In the search field, search for the name of the user. You can also filter users by role in the Roles drop-down menu.
Edit Role
Click the user's Options icon, then select the Edit Role link.
Edit Course Role
Click the Role drop-down menu, then select the new role for the user. 
Note: Depending on your user permissions, you may not be able to select from all user roles in your course.
Edit Multiple Roles
If a user has multiple roles in the course, selecting a new role overwrites all the user's current enrollments. 
Update Role
Click the Update button.
