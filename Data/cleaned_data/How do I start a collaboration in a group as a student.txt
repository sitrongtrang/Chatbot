How do I start a collaboration in a group as a student
Open Collaborations
In Group Navigation, click the Collaborations link.
Authorize Access
By default, the Collaborate using drop-down menu is set to Google Docs.
If you have not previously authorized a web tool access in Canvas, you may be prompted to authorize access. Click the Authorize Google Drive Access button.
Create Collaboration
To create a collaboration document using Google Docs, enter a document name in the Document name field, create a description, and click the names of the people you want to collaborate with. 
Verify Collaborators
Verify the users you want to collaborate with. 
To remove any user, hover your cursor over the user name and click the Remove icon. 
Start Collaborating
Click the Start Collaborating button.
