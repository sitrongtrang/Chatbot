How do I edit and remove hyperlinks in the Rich Content Editor as a student
Open Rich Content Editor
Open the Rich Content Editor when creating or editing a group announcement, discussion, or page, or when completing an assignment or quiz.
Note: The Rich Content Editor supports keyboard shortcuts. To view the Keyboard Shortcuts menu, click the Keyboard icon or press ALT+F8 (PC keyboard) or Option+F8 (Mac keyboard).
Edit Link
Click the link in the Rich Content Editor. Then click the Link Options link. 
You can also click the Link Options arrow and select the Edit Link option.
Note: To view the Links icon, you may have to click the Options icon. 
View Link Options
In the Link Options menu, you can edit the hyperlinked text in the Text field. You can edit the URL in the Link field. 
To save your changes, click the Done button. 
Note: When clicked, external hyperlinks automatically open in a new browser tab.
Remove Link
To remove a link, click the Link Options arrow and select the Remove Link option. 
Remove Multiple Links
To remove multiple links, highlight the text that contains the links you want to remove. Then click the Link Options arrow and select the Remove Links option.
Save Changes
Click the Save button.
Note: When using the Rich Content Editor in Discussions, the Save button may appear as the Post Reply button.
