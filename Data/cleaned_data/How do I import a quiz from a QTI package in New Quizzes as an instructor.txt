How do I import a quiz from a QTI package in New Quizzes as an instructor
Open Quizzes
In Course Navigation, click the Quizzes link.
Note: You can also access quizzes from the Assignments page.
Add Quiz
Click the Add Quiz button.
Create New Quiz
In the Choose a Quiz Engine screen, select the New Quizzes option. Then click the Submit button. 
Edit Quiz Details
Edit the details for your quiz.
Save Quiz
To save quiz details, click the Save button. To open the New Quizzes build page and import content, click the Build button.
Import Content
Your quiz will open to the Build page. To import content, click the Options icon and select the Import Content option.
Import QTI File
To import a QTI file, drag and drop the file or select a file by clicking the Browse button.
To start the import, click the Import button.
View Import Status
A status bar will display the status of your import.
View Quiz
View your imported quiz.
