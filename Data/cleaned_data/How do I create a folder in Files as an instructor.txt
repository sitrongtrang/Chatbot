How do I create a folder in Files as an instructor
Open Files
In Course, User, or Group Navigation, click the Files link.
Add Folder
Click the Add Folder button.
Name Folder
Type the name of the folder in Name field. To save the name of the file, press Return (Mac) or Enter (PC) key or click the Check button. To return to the previous title, click the Cancel button.
View Folder
Your folder will appear in both panes of the Files view. You can organize your files on Canvas by moving them to folders.
Delete Folder
To delete a folder, click the line item for the folder and then click the Delete icon. Or click the Options icon and then click the Delete link.
Note: Deleting a folder will also delete all files in that folder.
