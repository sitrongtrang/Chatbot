How do I embed media files from Canvas into the Rich Content Editor as a student
Open Rich Content Editor
Open the Rich Content Editor when using one of the Canvas features which support the Editor.
Note: The Rich Content Editor supports keyboard shortcuts. To view the Keyboard Shortcuts menu, click the Keyboard icon or press ALT+F8 (PC keyboard) or Option+F8 (Mac keyboard).
Upload Media from Toolbar
Alternatively, you can embed your media files from the toolbar. Click the Media icon. To view images saved in your Canvas user files, click the User Media option. 
To view the Record/Upload Media icon, you may have to click the Options icon.
Note: If the Media options do not display in the toolbar, media uploads have been disabled by your institution.
Upload Media from Menubar
Click the Insert link. Navigate to view the Media options, and select the User Media option. 
Select Media File
By default, when embedding a media file in the Rich Content Editor, the Add menu displays media files from your selected source. To change the media file source, click the Source drop-down menu. 
To sort media files by date added or alphabetically, click the Sort by drop-down menu. To search for a media file, enter the file name in the Search field. You must enter at least three characters for search results to display.
Click the media file to embed.
Note: The file will flash before it embeds in the Rich Content Editor. 
View Embedded Media
View your embedded media file in the Rich Content Editor.
Save Changes
Click the Save button.
Note: When using the Rich Content Editor in Discussions, the Save button may appear as the Post Reply button.
View Content
View the content created in the Rich Content Editor.
