How do I view login and logout activity for a user in an account as an admin
Open Account
In Global Navigation, click the Admin link, then click the name of the account.
Open Admin Tools
In Account Navigation, click the Admin Tools link.
Open Logging
Click the Logging tab.
Select Log Type
From the log type drop-down menu, select the Login / Logout Activity option.
Search for User
Type the name of the user in the search field, then click the Find button. Click the user's name. 
Enter Date Range
Create the date range you want to search. You can enter dates in the From Date and To Date date fields, or you can use the calendar icon to select dates. Then click the Find button.
Find Logging Activity
Click the Find button.
View Logging Activity
View the login/logout activity for the user.
