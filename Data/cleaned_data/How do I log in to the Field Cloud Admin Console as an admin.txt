How do I log in to the Field Cloud Admin Console as an admin
Log in to Canvas
Using your institution's Canvas URL, enter your Canvas credentials and log in to your account.
Open Console
To log in to the Field Admin Console, enter adminconsole.canvaslms.com in the URL field in a new browser window or tab.
Select Account
If you have an account with more than one Canvas URL, you may be asked to select the account that you want to use to log in. 
View Console
View the Field Admin Console for your account. Learn how to use the Field Admin Console.
